# Global Mental Health Disorder Trends

This project uses SQL and Python to analyze global mental health prevalence data by country and year.

## Tools
- DuckDB / SQL
- Python
- pandas
- matplotlib

## Key Questions
- Which countries have the highest average depression prevalence?
- Are high rates driven by isolated spikes or consistent long-term patterns?

## Key Findings
- Greenland, Morocco, and Lesotho had the highest average depression prevalence in the dataset.
- Greenland remained consistently elevated across the observed years.
- Morocco peaked in the early 2000s before gradually declining.
- Lesotho showed a steady upward trend over time.

## Files
- `MH_Country_Analysis_for_Depression.sql` — SQL cleaning and analysis queries
- `top3_depression_trend.csv` — exported SQL result for charting
- `top3_depression_chart.png` — trend visualization
